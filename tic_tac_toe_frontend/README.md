# Tic Tac Toe (React Frontend)

A self-contained **two-player (local)** Tic Tac Toe web app with a **retro-themed** UI.

## Features

- 3Ã—3 clickable board
- Two-player local play (X / O), alternating turns
- Win detection (rows, columns, diagonals)
- Draw detection
- Highlights the winning line
- Restart/reset button
- No backend dependency (uses internal React state only)

## Run locally

From this folder:

```bash
npm install
npm start
```

Then open: http://localhost:3000

## Notes

- UI is light-themed with accent colors:
  - Primary: `#3b82f6`
  - Accent: `#06b6d4`
- The game disables further moves after a win/draw until you restart.
